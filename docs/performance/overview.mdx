## ARC mental model

```swift
class ImageDownloader {
    func fetch(_ url: URL, completion: @escaping (UIImage?) -> Void) {
        URLSession.shared.dataTask(with: url) { [weak self] data, _, _ in
            // self is captured weakly; if released, callback no-ops
            self?.cache(data)
            completion(data.flatMap(UIImage.init(data:)))
        }.resume()
    }
}
```

Closures capture `self` strongly by default. Use `[weak self]` or `[unowned self]`
to break retain cycles. The Memory Graph debugger (Xcode → Debug → View Memory Graph)
shows exactly which objects survive when they shouldn't.

<Callout kind="tip" title="`unowned` vs `weak`">
Use `unowned` only when the closure cannot outlive `self`. Otherwise default to
`weak self` — it's safer and the optional unwrap cost is negligible.
</Callout>

## MetricKit — production telemetry from Apple

```swift
final class MetricsObserver: NSObject, MXMetricManagerSubscriber {
    func didReceive(_ payloads: [MXMetricPayload]) {
        for p in payloads {
            // Disk writes, hangs, app launch metrics
            log("hangs: \(p.applicationResponsivenessMetrics?.histogrammedApplicationHangTime)")
        }
    }
}
```

Subscribe in `application(didFinishLaunchingWithOptions:)` and forward to
your analytics / logging backend. Real metrics from real users, no SDK required.

## Common wins, ranked by impact

1. **Defer SDK initialization** to after the first frame.
2. **Use compositional collection layouts** instead of nested stack views.
3. **Precompute formatted strings** (`DateFormatter` is expensive — cache it).
4. **Use `Image(uiImage:)` with downsampling** for large photos in lists.
5. **Adopt baseline diffable data sources** — they're significantly faster than reload.

## Key takeaways

- **Profile before optimizing** — Instruments and the Memory Graph are your truth source.
- **`weak self` is a discipline**, not an exception.
- **Launch < 400ms** on the lowest-spec supported device — measure, then defer.
- **MetricKit is free production telemetry** — wire it on day one.
